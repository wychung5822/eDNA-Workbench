# Phylogenetic Viewer
A tool for visualizing phylogenetic trees.

**Input:**
*   **Tree File:** A **Newick** format file, with or without branch values (e.g., `tree.nwk`).

**Output:**
*   **Middle box:** Tree visualization with branch length at the top.
*   **Left box:** Search and export files
*   **Right box:** Control panel

**Configuration:**
*   **Visualization:** View the tree structure.
*   **Node Management:** Tick to show internal node names; click internal nodes to collapse/reduce subtrees; rename nodes.
*   **Resizing:** Resize the view horizontally or vertically.
*   **Collapse trees by certain branch length:** Users can click on the branch length scale to automatically merge subtrees.
